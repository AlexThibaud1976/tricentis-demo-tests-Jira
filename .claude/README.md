# Claude Configuration Directory

This directory contains configuration files and resources to optimize interactions with Claude Code for this Playwright/BrowserStack test automation project.

## Directory Structure

```
.claude/
├── README.md              # This file
├── prompts/               # Reusable prompt templates
├── snippets/              # Code snippets for common patterns
├── workflows/             # Common workflow guides
└── context/               # Project-specific context files
```

## Purpose

The `.claude` directory helps Claude Code understand project-specific patterns, common tasks, and best practices without needing to explain them each time.

## Usage

Files in this directory are automatically available to Claude Code and help with:
- Understanding project architecture
- Following established patterns
- Executing common workflows
- Maintaining consistency across changes
